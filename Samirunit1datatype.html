Object And Classes


1. The main purpose of C++ programming is to add object orientation to the C programming language and classes are the central feature of C++ that support object-oriented programming and are often called user defind types.
2. A class is used to specify the form of an object and it combines data representation and methods for manipulating that data into one package.
The data and function within a class are called members of the class.


Object


1. Identity The name associated with an object helps in identifying the object i.e. what we call the object as? For example a wall clock is the name of object called clock which is hanged on the wall.

2. State An object can be in many state i.e. a computer can be in following state (i) on state (ii) off state (iii) Out of order state.

3. Behaviour What the object does ? Or what it is capable of doing ?

Example : a person can sit, stand, sleep, walk, and talk.
These object have been classified as belonging to a group or class. On the basis of commonality of possible states and behaviour we conclude from the above discussion that the object can be described as the possible common state, identity and its behaviour.
