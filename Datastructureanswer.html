Unit 3

SAQ

1.Linked list(Define & diagram)?
Ans: A linked list is a linear data structure consisting of a sequence of elements called nodes
Each node contains two parts: data and a reference (or pointer) to the next node in the sequence.

2.Singly Linked list (Define & diagram)
Ans: A singly linked list is a data structure consisting of nodes where each node contains two parts: data and a pointer to the next node in the sequence. The first node is called the head, and the last node points to null, indicating the end of the list.

3.Doubly Linked list(define & diagram)?
Ans: A doubly linked list is a data structure where each node contains three parts: data, a pointer to the previous node, and a pointer to the next node. This structure allows traversal in both forward and backward directions.

4.Circular List(Define & diagram)
Ans: A circular linked list is a type of linked list where the last node points back to the first node, forming a loop. This means that traversal can start from any node and reach all other nodes in the list.

5.Hashing(Define & diagram)
Ans: Hashing is one of the searching techniques that uses a constant time. The main idea behind the hashing is to create the (key/value)pairs. If the key is given, then the algorithm computes the index at which the value would be stored.
