Tokens in c++ are the smallest individual element that is meaningful to the compiler

    The tokens are classified into:
  • Keywords
  • Identifiers
  • literals
  • Punctuators/ seperators
  • Operator
    • Keywords
    These are reserved words that have a specific meaning in the language, such as int, float, while, and if.
    Identifiers
    These are names given to variables,function or ther objects in code
    Operator
    These are symbols to represent such as +,-,* and/
    Punctuators
    There are symbols that are used ot seperate tokens such as (,), ;, and ,
    Literals
    Literals in C++ are data that represent fixed values. They can be used directly in the code without being assigned to a variable. Literals cannot be changed once they are created.
    There are five types of literals in C++:
    Integer literals: represent whole numbers, such as 10, 25, and -30.
    Floating-point literals: represent numbers with decimal points, such as 3.14159 and -2.71828.
    Character literals: represent single characters, such as 'a', 'B', and '$'
    String literals: represent sequences of characters, such as "Hello, world!" and "This is a string.".
    Boolean literals: represent true or false values, such as true and false.

    Tokens are usually separated by white space, which can be one or more blanks, tabs, or newlines. The C++ standard specifies a basic source character set that may be used in source files.
