In the OSI reference model, the communications between a computing system are split into seven different abstraction layers:

  • Data Link

  • Transport
  • Session

    1.Explain about OSI layers in detail?

    Ans. The Open Systems Interconnection (OSI) model describes seven layers that computer systems use to communicate over a network.
    It was the first standard model for network communications, adopted by all major computer and telecommunication companies in the early 1980s.

  • Physical Layer: The physical layer is responsible for the physical cable or wireless connection between network nodes.
  • The physical layer handles the transmission of raw bits over a communication link
  • Data Link LayerThe data link layer then collects a stream of bits into a larger aggregate called frame. The frames are delieverd to the host
  • Network Layer:Receives frames from the data link layer and delivers them to their destination
  • Transport layer:Then transport layer implements process to process channel. Here the unit of data exchange is called a message rather than a pakcet or a frame
  • Session Layer:The session layer creates communication channels, called sessions, between devices.
  • Opens and closes communication between two devices, and controls conversations between different computers
  • Presentation LayerIt defines how two devices should encode, encrypt, and compress data so it is received correctly on the other end.
  • Translates application data into a network format and prepares data for the application layer
  • Application layer:Application layer protocol includes things like protocol (HTTP), which is the basis of the World Wide Web
  • Establishes communication between the application and the end user

    • 2. Explain about CRC technique with an example?

    CRC stands for Cyclic Redundancy Check.
    It is based on binary divison, a sequence of redundant bits, called cyclic redundancy check bits.
    Cyclic codes are used for error detection by link layer protocols where the data transport unit is called a frame.

      Steps to perform crc divison are
    1. Find the length of divisor 'L'
    2. Appeal 'L-1' bits to the orignal message
    3. Perform binary divison operation
    4. Reaminder of division = CRC number

    • 4.Explain about CSMA/CD algorithm?

  • The transmitting station determines wethere the channel is busy or idle with a frame is ready
  • If the channel is occupied, the station waits till it is free.
  • The station begins transmitting and continuously scans the channel for collisions if the channel is empty
  • The station begins the collision resolution mechanism when a collision is detected.
  • After finishing frame transmission, the stations reset the retransmission counters.

    What is CSMA/cd?

    Carrier sense multiple access with collision detection (CSMA/CD) is a medium access control (MAC) protocol that specifies how network devices should react when two devices try to use the same data channel at the same time
