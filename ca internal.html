    • 2. Explain about bus structure?

  • In computer architecture, a bus is a communication system that transfers data between components within a computer or between computers. It's also known as a data highway or databus

  • A bus structure is a set of conductors that carry data, address, and control signals within a computer system. It acts as a communication highway connecting the central processing unit (CPU), memory, input/output devices, and other peripherals.
  • A system bus usually consists of a range of distinct lines, typically numbering from fifty to hundreds. Each line is designated for a specific function, and these lines can be divided into three main functional categories: data lines, address lines, and control lines.
  • Multiple components or devices can use the same bus structure to transmit information signals to each other. However, only one pair of devices can use the bus at a time to communicate successfully.
  • Most computer have three busses:
  • Control bus
  • Data Bus
  • Address Bus

    • 3. Explain about memeory location and address?

  • Memory location:- Main memory is second major subsystem in a computer
  • Data is transferred to and from memory in groups of bits called words
  • The memory consists of many millions of storage cells, each of which can store a bit of information having value 0 or 1
  • Bits are seldom handled individually. The usual approach is to deal with them in groups of fixed size
  • Memory is organised so that a group of n bits can be stored or retrieved in a single, basic operation
  • Each group of n bits is referred to as a word of information, and n is called the word length
  • Modern computers have word lengths that typically range from 16 t6 64 bits
    • Each memory location has an address, and each address refers to a single memory location.

    There are two types of memory addresses:

  • Physical addresses: These are codes that the CPU or other device can use to access the corresponding memory location.
  • Logical addresses: These are virtual addresses that application programs use

    • 4. Explain in detail about basic input output operations?

    Input/output (I/O) operations are a type of communication process in computing that involves transferring data between a computer and its external environment.

  • Input: Involves reading data from an external source into the computer's memory.
  • This data can come from users' input devices, such as keyboards, mice, or touchscreens.
    It can also come from other devices or sensors connected to the computer, such as scanners, cameras, or microphones.
  • Basic Input Output System (BIOS): Handles input and output operations at a basic level. BIOS helps provide services to the operating system,
    such as booting up and initializing of hardware
  • I/O devices: Can be categorized as storage, networking, user interface, or related hardware-enabled services.
    Some common I/O devices include printers, hard disks, keyboards, and mice.
