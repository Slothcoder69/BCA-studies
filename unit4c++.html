Inheritance

Inheritance:

Inheritance: The capablility of a class to derive properties and characterstics from another class is called Inheritance. Inheritance is one of the most import features of OOP's. Inheritance is the process of creating new class called derived class from the existing class called the base class

One of the main advantages of inheritance is that reusability of the code

Imporance of Inheritance

  • 1.It drastically reduces the code redundancy as it allows the derived class to inherit all the properties of base class, leaving no room for duplicate data to achieve the same task
  • 2.Inheritance in c++ offers the feature of code reusability. We can use the same fragment of code multiple times. To sum it up, inheritance helps us save our development time, maintain data in simplified manner and gives the provision to make our code extensible
  • 3.It increases the code reliability by providing a definite body to the program
  • 4.The transitive nature of inheritance makes things easier for us

    • Types of Inheritance:

  • 1.Single Inheritance
  • 2.Multilevel Inheritance
  • 3.Multiple Inheritance
  • 4.Hierarchial Inheritance
  • 5.Hybrid Inheritance

    • Singel Inheritance: A derive class with only 1 base class is called single inheritance

    Multilevel Inheritance: A derived class with one base class and that base class is derived class of another is called Multilevel Inheritance

    Multiple Inheritance: Derived class with multiple base class is called multiple inheritance

    Hierarchial Inheritance: Multiple derived classes with same base class is called hierarchical inheritance

    Hybrid Inheritance: Combination of multiple and hierarchical inheritance is called hybrid inheritance

    Operator Overloading:


    Operator Overloading:     Operator overloading allows operators to have different meanings based on the context, especially for user-defined types. It lets you define how operators like + or * work with objects of custom classes, making code more intuitive.

    Operator Overloading is used to perform operation on user defined data type

    Any operator can be overloaded in C++, however there are few operators which can note be overloaded in c++. they are:

  • Scope Operator
  • sizeof
  • member operator
  • memeber pointer selectpr
  • ternary operator

    • Operator Overloading can be done by implementing a function which can be

  • Member function
  • Non-Memeber function
  • friend function

    • Rules for Overloading operators

  • Only Existing Operators Can Be Overloaded: You cannot create new operators. You can only overload existing ones.
  • Certain Operators Cannot Be Overloaded: The following operators cannot be overloaded: ::, .:, .*, ?:, sizeof, typeid, alignof.
  • Overloaded operators must have at least one operand that is a user-defined type
  • The overloaded operators can be member functions or global functions.

    • Restrictions on Operator Overloading

  • Precedence and associativity: Overloading an operator cannot change its precedence and associativity.
  • Arity: Overloading an operator cannot change its arity, the number of operands it takes.
  • New operators: You cannot create new operators, only overload existing ones.
  • Built-in types: You cannot change the meaning of an operator for built-in types.
  • Overloading for scope resolution: The scope resolution operator (::) cannot be overloaded.

    • Operator Overloading can be done by using two important apporaches, they are Overloading Unary Operator and Overloading Binary Operator

    Overloading Unary Operator: In unary operator function, no arguments should be passed. It works only with one class objects.It is a overloading of an Operator operating on a single operand.

    Note: Unary operators include the increment (++) and decrement (--) operators.

    Overloading Binary Operator: In binary operator overloading function, there should be one argument to be passed.

    Pointers

    a pointer is a variable that stores the memory address of another variable.
    Pointers are symbolic representation of addresses.
    Syntax:
    datatype*var_name;

    Features of pointers

  • 1.Pointers save memory space.
  • 2.Execution time with pointers is faster because data are manipulated with the address, that is, direct access to memory location
  • 3.Pointers are used with data structures
  • 4.Memory is accesssed efficently with pointers. The pointer assigns and releases the memory as well. Hence it can be said the Memory of pointers is dynamically allocated
  • 5.An array, of any type can be accesed with the help of pointers.
  • 6.pointers are used for file handling

    • Array of pointers

    An array of pointers is an array that consists of variables of pointers type, which means that the variable is a pointer addressing to some other element.
    Suppose we create an array of holding 5 integer pointers; then its declaration would look like.
    int*ptr[5];
    In the above declaration we declare am array of pointer named as ptr, and it allocates 5 integer pointer in memory
